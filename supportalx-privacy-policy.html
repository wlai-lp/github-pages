SupportalX — Privacy Policy

Last updated: February 8, 2025

Overview

This is an internal tool for LivePerson resource only. SupportalX is a Chrome extension that streamlines LivePerson support workflows by automating login processes. Your privacy is important to us. This policy explains what data the extension handles and how it is used.

Data We Collect

SupportalX stores the following data locally in your browser using Chrome's built-in storage API (chrome.storage.sync):

  • Username — Your LivePerson admin username
  • Password — Your LivePerson admin password
  • Site IDs — LivePerson account site IDs you enter
  • Favorites — Saved site IDs with optional aliases
  • Settings — Your last saved timestamp

How Your Data Is Used

Your stored credentials are used exclusively to auto-fill login forms on LivePerson's own websites (liveperson.net). Specifically:

  • LP Login — submits credentials to LivePerson's LiveEngage console
  • LPA Admin Access — submits credentials to LivePerson's admin access portal
  • Houston Login — submits credentials to LivePerson's backoffice portal

This is functionally identical to you manually typing your credentials into these login pages.

Where Your Data Is Stored

All data is stored locally in your browser using chrome.storage.sync. This means:

  • Data is stored on your device and synced across your Chrome browsers via your Google account
  • Data is encrypted in transit by Chrome's sync infrastructure
  • Data is never stored on any server owned or operated by the extension developer

Data Sharing

SupportalX does not:

  • Send your data to any third-party servers
  • Share your data with any third parties
  • Sell your data
  • Collect analytics or telemetry
  • Use tracking scripts or cookies
  • Load or execute any remote code

Network Requests

The extension makes the following network requests:

  • LivePerson API (api.liveperson.net) — to resolve the correct admin domain for a given site ID. Only the site ID is sent; no credentials are included in this request.
  • LivePerson login endpoints (*.liveperson.net) — to submit login forms with your credentials, identical to a manual browser form submission.

Permissions

The extension requests only the permissions necessary for its functionality. A full explanation of each permission is available on the Chrome Web Store listing.

Data Deletion

You can delete all stored data at any time by:

  • Clearing the username and password fields in the extension's settings and clicking Save
  • Removing the extension from Chrome (this deletes all associated storage)

Changes to This Policy

If this privacy policy is updated, the "Last updated" date at the top will be revised. Continued use of the extension after changes constitutes acceptance of the updated policy.
